# medical-computational-vision-
Medical advanced computational vision project for MRI DICOM analysis, reconstruction and machine learning.

Genom att dekonstruera hjärnans biologi med ingenjörsvetenskaplig systematik, transformerar detta projekt ostrukturerad 3T MRI-data till intelligenta, sökbara IFC-modeller

Detta projekt demonstrerar en tvärvetenskaplig pipeline där ostrukturerad medicinsk bilddata (3T MRI) transformeras till en intelligent, sökbar och skalbar digital tvilling. Genom att låna principer från byggbranschens BIM-logik (IFC), finansvärldens riskanalys och musikproduktionens signalbehandling, skapar vi ett system där varje biologisk detalj behandlas som ett strukturerat informationsobjekt.

Tekniska Moduler (Deep Dive)
1. Signal Processing & Physics-Informed AI
Här bryter vi ner rådata från 3T-skanningar. Istället för att bara se pixlar, analyserar vi signalen bakom.

Fourier Synthesis: Med erfarenhet från digital musikproduktion (FL Studio) tillämpas Fast Fourier Transforms (FFT) för att isolera 3T Larmor-frekvenser och minimera brus.

Precision Enhancement: Algoritmer som möjliggör analys ner till 0,1 mm, vilket krävs för att identifiera mikro-anomalier som traditionella system missar.

Bayesian Uncertainty: Integration av en sannolikhetsmodell som ger varje detekterad punkt ett "confidence score", inspirerat av finansiell kvantanalys.

2. Computational Geometry & Fractals
Biologi följer sällan raka linjer. Därför använder vi icke-linjär matematik:

Fractal Dimensioning (FD): Vi beräknar vävnadens komplexitet (Hausdorff-dimension) för att matematiskt skilja patologisk tillväxt från friska nervbanor.

Manifold Learning: Reduktion av högdimensionell data för att hitta underliggande mönster i hjärnans struktur.

Structural Integrity Analysis: Nervbanor analyseras som bärande element i en konstruktion (Skeletonization), där vi mäter vridmoment och avvikelser.

3. Spatial Systems Architecture (BIM/IFC)
Här sker "uppskalningen". Vi konverterar medicinsk data till en global industristandard.

IFC Schema Mapping: Varje anomali klassificeras som en IfcBuildingElementProxy med unika GlobalID.

Custom Property Sets (Psets): Injektion av kliniska metadata (t.ex. Pset_AnomalyDynamics) direkt i 3D-objektet.

Queryable Anatomy: Möjliggör SQL-liknande sökningar i en hjärna: "Visa alla objekt med Fractal Dimension > 1.7 och Bayesian Confidence > 90%".
